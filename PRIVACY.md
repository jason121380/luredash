## 1. 適用範圍

本隱私權政策(以下稱「**本政策**」)說明「**LURE Meta Platform**」(以下稱「**本服務**」、「**我們**」、「**平台**」)如何蒐集、使用、保存、分享及保護您的個人資料及相關使用紀錄。

本政策適用於:
- 透過瀏覽器使用 https://metadash.zeabur.app(或本服務之其他正式網域)的所有使用者。
- 經由本服務的公開分享連結(`/r/<活動ID>`)瀏覽報告的「**報告觀看者**」。
- 透過 LINE 群組接收由本服務發送之自動推播訊息的「**訊息接收者**」。

「**個人資料**」之定義依照中華民國《個人資料保護法》(以下稱「個資法」)第 2 條第 1 款。

## 3. 我們蒐集哪些資料

### 3.1 您主動提供的資料

#### 3.1.1 Facebook 登入時提供的資料
當您透過 Facebook 登入授權本服務存取時,我們會自 Meta(Facebook)取得:

- 您的 **Facebook 使用者 ID**、**姓名**、**個人頭像 URL**
- 您的 **Facebook 長期存取權杖(access token)**,有效期約 60 天
- 您在 Facebook 上擁有存取權限的 **廣告帳戶清單**、**Business Manager 清單**、**粉絲專頁清單**

授權範圍對應之 Facebook 權限:`ads_read`、`ads_management`、`business_management`、`pages_read_engagement`。

#### 3.1.2 您建立的 LINE 官方帳號(OA)資料
若您在「LINE 推播設定」中新增 LINE Official Account,我們會儲存:

- **LINE Channel Secret**
- **LINE Channel Access Token**
- 您為該 OA 自定義的顯示名稱
- 我們向 LINE API 取得的該 OA 真實顯示名稱(displayName)

#### 3.1.3 訂閱與付款資料
當您訂閱付費方案時,**金流由 Polar.sh 處理**,我們**不會接觸您的信用卡 / 銀行帳戶資料**。我們會儲存 Polar.sh 提供的:

- Polar 客戶 ID(`polar_customer_id`)
- Polar 訂閱 ID(`polar_subscription_id`)
- 您的方案層級、訂閱狀態、扣款週期、是否已排程取消
- Polar 推送的訂閱事件(billing_events,例如「訂閱建立」「扣款成功」「扣款失敗」)

#### 3.1.4 您在平台上的設定操作
- 您所選擇要顯示的廣告帳戶清單
- 您的廣告帳戶排序
- 行銷活動暱稱(店家、設計師)— **此項為團隊共用**
- 費用加成設定、釘選清單、預設加成比例 — **此項為團隊共用**
- LINE 推播設定(推播頻率、時間、推播報告欄位、優化建議開關等)
- AI 幕僚的執行紀錄(時間、所選帳戶)

### 3.2 自 Facebook 取得的廣告數據

當您使用儀表板、分析、警示、財務、AI 幕僚等功能時,本服務會透過 Facebook Marketing API 即時取得:

- 廣告帳戶的整體成效數據(花費、曝光、點擊、CTR、CPC、CPM、頻次、觸及)
- 行銷活動 / 廣告組合 / 廣告層級的元資料與成效
- 廣告素材(縮圖、文案、影片、互動數)
- 受眾洞察(版位、性別、年齡層、地區的成效切分)
- 私訊起始數、購買數、加入購物車數、連結點擊、ROAS 等行動指標

這些資料**不會由本服務主動儲存**,僅在您發出請求時即時從 Facebook 取得並以記憶體快取暫存(預設 5 分鐘)。

### 3.3 自 LINE 取得的資料

- LINE 群組 ID、群組顯示名稱(透過 `/v2/bot/group/{id}/summary` 取得)
- LINE Webhook 事件(bot 加入 / 離開群組、收到的訊息事件)
- LINE 官方帳號的本月推播使用量(`/v2/bot/message/quota` 即時查詢時)

### 3.4 推播紀錄(`line_push_logs`)

每次本服務代您發送 LINE 推播時,我們會記錄:

- 推播時間、目標 LINE 群組 ID、配置 ID
- 訊息預覽(`altText`,前 200 字元)
- 成功與否、若失敗則記錄錯誤訊息

### 3.5 瀏覽器本地儲存(localStorage)

僅為提升您的使用體驗,以下資料儲存於您自己的瀏覽器,**不會上傳到本服務的伺服器**:

- `fb_active_accounts`:目前儀表板選中的廣告帳戶
- `filter_active_only`、日期選擇器偏好、側邊欄收合狀態
- `meta_dash_fb_token`:Facebook 登入快取(瀏覽器自行管理)
- 各種 UI 偏好(分頁、排序、欄位顯示等)

### 3.6 我們**不**蒐集的資料

- 您 Facebook 的好友清單、私訊內容、個人貼文
- 您 LINE 群組內成員的個人資料(我們僅知道群組 ID,無法取得成員清單)
- 您 LINE 群組內成員傳送的訊息內容(僅 bot 接收事件會被記錄)
- 您的瀏覽歷史、跨網站追蹤資料、IP 地理定位

## 6. 資料保留期限

| 資料類別 | 保留期限 |
|---|---|
| FB 存取權杖(`user_fb_tokens`) | 至 token 自然過期(約 60 天)或您主動登出 |
| 使用者偏好設定(`user_settings`) | 至您要求刪除為止 |
| LINE 官方帳號憑證(`line_channels`) | 至您主動刪除該 channel 為止 |
| LINE 群組記錄(`line_groups`) | bot 退出後仍保留紀錄(`left_at` 標記),為審計目的;您可主動要求清除 |
| LINE 推播紀錄(`line_push_logs`) | 12 個月(以資料庫實際大小評估,可能更短) |
| AI 幕僚執行紀錄(`agent_advice_runs`) | 12 個月(用於計算月配額) |
| 訂閱與付款紀錄(`subscriptions`、`billing_events`) | 依會計法規,7 年 |
| Polar.sh 完整付款記錄 | 由 Polar.sh 保管,期限依其政策 |
| 行銷活動暱稱、財務加成設定(團隊共用) | 至團隊管理者主動清除為止 |
| 伺服器存取紀錄(log) | 30 天 |

帳戶刪除後,我們會在合理時間內(目標 30 天內)從生產資料庫永久刪除您的個人資料,但下列情況除外:
- 法律規定必須保留的資料(如稅務憑證)
- 為防止詐欺、安全事件追溯所需之最小範圍紀錄
- 已匿名化、無法識別個人之統計資料

## 7. 資料安全

我們採取下列技術與管理措施保護您的資料:

- 全站 HTTPS / TLS 1.2+ 加密傳輸
- 資料庫位於 VPC 內,僅允許應用伺服器存取
- 存取憑證透過環境變數注入,**不存放於版本控制系統**
- LINE Webhook 採 HMAC-SHA256 簽章驗證(`X-Line-Signature`)
- Polar Webhook 採簽章驗證(`Webhook-Signature` 規範)
- 應用層權限分層:
  - 廣告帳戶資料**按使用者隔離**(每位使用者僅能看到自己 Facebook 帳號授權的廣告帳戶)
  - LINE 官方帳號採「擁有者 + 共享授權」模型,需擁有者明確邀請對方才能存取
  - 共享授權支援「管理員 / 唯讀」兩種權限層級

**現階段已知的安全限制**(我們持續改善中):
- 部分內部 API 端點依賴使用者透過 query string 傳入 `fb_user_id`,尚未導入完整的伺服器端簽章 session
- FB / LINE 存取權杖目前以明文形式儲存於資料庫,**計畫於後續版本導入 Fernet 加密**

我們**無法保證絕對安全**。如您發現安全漏洞,歡迎通報至上述聯絡信箱。

## 8. 您的權利

依個資法第 3 條(以及如適用,GDPR 第 15–22 條),您擁有以下權利:

1. **查詢及請求閱覽**:可請求我們提供您留存在本服務的個人資料副本
2. **請求製給複製本**:可請求機器可讀格式之副本(JSON / CSV)
3. **請求補充或更正**:資料有誤時可請求修改
4. **請求停止蒐集、處理或利用**:可暫停或限制特定處理目的
5. **請求刪除**:符合「資料蒐集目的消失」等法定情況時,可請求永久刪除
6. **撤回同意**:您隨時可登出本服務、刪除 LINE OA 整合、取消訂閱

**行使方式**:寄信至上述聯絡信箱,主旨註明「個人資料權利行使」,並提供能識別您身份的資訊(Facebook user ID 或註冊信箱)。我們將於收到請求後**30 日內**回覆。

## 9. 公開分享連結(`/r/<活動ID>`)

當您將分享連結傳給第三方查看廣告報告時:

- **觀看者不需要登入**任何系統即可瀏覽
- 該連結僅顯示**指定行銷活動**之成效摘要,不會洩漏您的 Facebook 帳號其他資料
- 連結預設**不過期**,但您可隨時透過刪除 LINE 推播設定間接停用相關連結
- 我們不會記錄個別觀看者的身份或瀏覽行為

如您不希望某個報告繼續被分享,請刪除對應的 LINE 推播設定或將該行銷活動下架。

## 10. AI 幕僚(AI 優化建議)資料處理

當您啟用 AI 幕僚功能時:

- 我們會將**選定帳戶/活動的成效統計摘要**(花費、私訊、CTR、CPC 等彙總數字)傳送至 Google Gemini API
- **不會**傳送您的 Facebook 存取權杖、客戶聯絡資料、廣告受眾原始資料
- Google 依其條款**不會將您的請求用於模型訓練**(以 Google 條款為準)
- 您可隨時不使用此功能;不使用此功能不會影響其他功能

## 11. LINE 推播訊息接收者

若您是收到本服務經由 LINE 群組發送之推播訊息的接收者:

- 您的個人資料**不會被本服務蒐集** — 我們僅知道群組 ID 與 bot 加入 / 離開事件
- 您可請該群組之管理員將 bot 移出群組,即可停止接收後續推播
- LINE 平台對推播訊息有其自身的計費與儲存規範(請參考 LINE 隱私權政策)

## 12. 跨境資料傳輸

由於本服務使用之第三方服務 / 主機代管商可能位於台灣以外地區(美國、日本、新加坡、歐盟等),您的資料可能涉及跨境傳輸。我們選擇之服務供應商均聲明遵循其所在司法管轄區之資料保護法規(如 GDPR、CCPA),並提供標準合約條款(SCC)等保障。

## 13. 兒童

本服務**不針對未滿 18 歲之個人提供**。若您未滿 18 歲,請勿使用本服務或提供任何個人資料。如我們發現意外蒐集了未滿 18 歲使用者之資料,將盡速刪除。
